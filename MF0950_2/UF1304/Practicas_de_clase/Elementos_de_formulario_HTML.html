Elementos de formulario HTML

El elemento <input>

El elemento de forma más importante es el <input>elemento.

El <input>elemento se puede mostrar de varias maneras, dependiendo del type atributo.

Ejemplo: <input name="Nombre" type="text">

Si type se omite el atributo, el campo de entrada obtiene el tipo predeterminado: "texto".

El elemento <select>

define una lista desplegable :

Ejemplo

<select name="cars">
<option value="volvo">Volvo</option>
<option value="saab">Saab</option>
<option value="fiat">Fiat</option>
<option value="audi">Audi</option>
</select>

El elemento selecionado, define una lista desplegable

Los <option>elementos define una opción que puede seleccionarse.

De manera predeterminada, se selecciona el primer elemento de la lista desplegable.

Para definir una opción preseleccionada, añadir el selectedatributo para la opción:

Ejemplo <option value="fiat" selected>Fiat</option>

Valores visibles:

Utilice el atributo size para especificar el número de valores visibles:

Ejemplo

<select name="cars" size="3">
<option value="volvo">Volvo</option>
<option value="saab">Saab</option>
<option value="fiat">Fiat</option>
<option value="audi">Audi</option>
</select>

Valores de opciones visibles

Utiliza el atributo tamaño para especificar el número de valores visibles.



Permite selecciones múltiples : (puedes marcar todos, con opción teclado)

Utilice el multiple atributo para permitir al usuario seleccionar más de un valor:

Ejemplo:

<select name="cars" size="4" multiple>
<option value="volvo">Volvo</option>
<option value="saab">Saab</option>
<option value="fiat">Fiat</option>
<option value="audi">Audi</option>
</select>

El Elemento <textarea>

El elemento <textarea> define un campo de entrada de líneas múltiples ( un área de texto ):

Ejemplo

<textarea name="message" rows="10" cols="30">
El gato juega en el jardín.
</textarea>

El rowsatributo especifica el número visible de líneas en un área de texto.

El colsatributo especifica la anchura visible de un área de texto.

Así es como el código HTML anterior se muestra en un navegador:

También puede definir el tamaño del área de texto mediante el uso de CSS:

Ejemplo:

<textarea name="message" style="width:350px; height:100px;">
El gato juega en el jardín.
</textarea>

Elementos de formulario HTML 5

HTML5 añade los siguientes elementos de formulario:

  • <datalist>
  • <output>

Nota: Los navegadores no muestran elementos desconocidos. Nuevos elementos que no son compatibles con los navegadores antiguos no "destruir" a su página web.

HTML5 Elemento <lista de datos>

El elemento <datalist> especifica una lista de opciones predefinidas para un elemento <input>.

Los usuarios podrán ver una lista desplegable de las opciones predefinidas, ya que los datos de entrada.

El lista atributo del elemento <input> debe hacer referencia al idatributo del elemento <datalist>

Ejemplo

<form action="/action_page.php">
<input list="browsers">
<datalist id="browsers">
<option value="Internet Explorer">
<option value="Firefox">
<option value="Chrome">
<option value="Opera">
<option value="Safari">
</datalist> 
</form>

Tag Description

<optgroup> Defines a group of related options in a drop-down list
<option> Defines an option in a drop-down list
<button> Defines a clickable button

<output> Defines the result of a calculation
