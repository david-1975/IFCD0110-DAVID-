El formato de texto

 

El formato de texto, es el marcado que se aplica a un texto para añadirle datos de forma. Al texto resultante se le conoce como texto formateado, texto con estilos o texto enriquecido.

Encabezados de texto

Los motores de búsqueda utilizan los encabezados para indexar la estructura y el contenido de sus páginas web.

Los usuarios hojear sus páginas por sus títulos. Es importante utilizar los encabezados para mostrar la estructura del documento.

Los encabezamientos <h1>deben ser utilizados para los encabezados principales, seguido por encabezados <h2><h3>,que son menos importante

Nota: Usar encabezados HTML para sólo encabezados. No utilice encabezados para que el texto GRANDE o negrita.

Etiquetas que dan Formateo de elementos HTML

HTML también define especiales elementos para la definición de texto con un especial significado.

HTML utiliza etiquetas como <b>y <i>para el formato de salida, como negrita o cursiva texto.

Etiqueta Descripción Ejemplo Utilización
<b> Define un texto en negrita Este texto está en negrita <b>Este texto está en negrita</b>
<i> Define un texto en cursiva Este texto está en cursiva <i>Este texto está en cursiva</i>
<strong> Define un texto resaltado Este texto está resaltado <strong>Este texto está resaltado</strong>
<em> Define un texto enfatizado Este texto está enfatizado <em>Este texto está enfatizado</em>
<small> Define un texto más pequeño Este texto se muestra más pequeño <small>Esto texto es pequeño<small>
<sup> Define un texto en superíndice Este texto es superíndice <sup>Este texto es superíndice</sup>
<sub> Define un texto en subíndice Este texto es subíndice <sub>Este texto es subíndice</sub>
<ins> Define un texto insertado (agregado) Este texto es un texto insertado <ins>Este texto está insertado</ins>
<s> Define un texto tachado Este texto es un texto tachado <s>Este texto está tachado</s>
<u> Define un texto subrayado Este texto está subrayado <u>Este texto es un texto subrayado</u>
<del> Define un texto eliminado o borrado Este texto está tachado eliminado <del>Este texto está eliminado</del>
<mark> Define un texto marcado Este texto a sido marcado marcado <mark>Este texto está marcado</mark>

Nota: Los navegadores muestran <strong>como <b>, y <em>como <i>. Sin embargo, hay una diferencia en el significado de las siguientes etiquetas: <b>y <i>define el texto en negrita y cursiva, pero <strong>y <em>significa que el texto es "importante".

Etiquetas que separan el texto

La etiqueta <p>separa el texto, añadiendo automáticamente un espacio (margen) antes y después de cada elemento <p>. El navegador eliminará todos los espacios en blanco y las líneas adicionales cuando se visualiza la etiqueta <p>.

La etiqueta <br> es una etiqueta vacía, lo que significa que no tiene niguna etiqueta final. Inserta un salto de línea (una nueva línea) sin comenzar un nuevo párrafo.

Un texto dentro de una etiqueta <pre> se muestra en una fuente de ancho fijo (generalmente Courier), y conserva ambos espacios y saltos de línea.
