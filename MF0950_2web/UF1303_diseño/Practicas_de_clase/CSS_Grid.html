Diseño de cuadrícula (Grid Layout)

El Diseño de cuadrícula Módulo CSS ofrece un sistema basado en el diseño de la red, con filas y columnas, por lo que es más fácil diseñar páginas web sin tener que utilizar flotadores y posicionamiento.

display: grid

Un diseño de la red consiste en un elemento padre, con uno o más elementos secundarios.

Un elemento HTML se convierte en un contenedor de cuadrícula mediante el establecimiento de la displaypropiedad ala rejilla o en línea de la red .

  • La propiedad grid-template-columns define el número de columnas en su diseño de cuadrícula, y puede definir la anchura de cada columna.
  • La propiedad grid-template-rows define el número de filas en su diseño de cuadrícula, y puede definir la anchura de cada fila.

    sintaxis CSS

    La propiedad grid-template es una propiedad abreviada para las siguientes propiedades:

    • grid-plantilla-row
    • grid-plantilla-column
    • grid-plantilla-áreas

    grid-template: none|grid-template-rows / grid-template-columns|grid-template-areas|initial|inherit;

Grid-gap

Se puede ajustar el tamaño de la separación mediante el uso de una de las siguientes propiedades:

  • La propiedad grid-gap es una abreviación para la propiedades grid-columna-gap y los grid-fila-GAP
  • La propiedad grid-column-gap establece la distancia entre las columnas
  • La propiedad grid-row-gap establece la separación entre las hileras:
  • La propiedad grid-column-start - indica en que columna empieza el item de las cuadrícula.
  • La propiedad grid-column-end indica en que columna terminará el item de la cuadrícula. 
  • La  propiedad grid-row-start - indica en que fila empieza el item de las cuadrícula.
  • La propiedad  grid-row-end indica en que fila terminará el item de la cuadrícula.

CSS la Propiedad grid de plantilla

La grid-template propiedad es una propiedad abreviada para las siguientes propiedades:

sintaxis CSS

grid-template: none|grid-template-rows / grid-template-columns|grid-template-areas|initial|inherit;

 

  • La propiedad grid-template-rows define la altura de cada fila.
  • La propiedad grid-template-columns define el número de columnas en su diseño de cuadrícula, y se puede definir la anchura de cada columna.
  • La propiedad justify-content se utiliza para alinear toda la rejilla dentro del recipiente. (space-evenly, space-around, space-between, center, start, end)
  • La propiedad align-content se utiliza para verticalmente alinear toda la rejilla dentro del recipiente.(space-evenly, space-around, space-between, center, start, end)

La Propiedad grid-template-area

La propiedad grid-area puede ser utilizada como una propiedad resumida para la rejilla-fila de inicio, rejilla-columna-inicio, rejilla-fila de extremo y las columnas de gama rejilla propiedades.

Elementos de la cuadrícula con nombre pueden ser referidos a la grid-template-areaspropiedad del contenedor de cuadrícula.

CSS La propieda Grid-auto-column

La propiedad grid-auto-columns establece un tamaño para las columnas en un recipiente de rejilla.

Esta propiedad sólo afecta a las columnas con el tamaño no establecido.

sintaxis CSS

grid-auto-columns: auto|max-content|min-content|length;

auto Valor por defecto. El tamaño de las columnas está determinado por el tamaño del contenedor.
fit-content()
max-content Establece el tamaño de cada columna en función del elemento más grande de la columna
min-content Establece el tamaño de cada columna en función del elemento más pequeño de la columna
minmax(min.max) Establece un rango de tamaño mayor o igual a min y menor o igual a max
length Establece el tamaño de las columnas, utilizando un valor de longitud legal.
% Establece el tamaño de las columnas, usando un valor de porcentaje

CSS La propiedad grid-auto-flow

La propiedad grid-auto-flow controla el número de elementos de auto-colocada se insertan en la red.

sintaxis CSS

grid-auto-flow: row|column|dense|row dense|column dense;
Value Description
row Valor por defecto. Coloca artículos llenando cada fila.
column Coloca los elementos rellenando cada columna.
dense Coloca objetos para rellenar cualquier agujero en la rejilla.
row dense Coloca los elementos rellenando cada fila y rellena los orificios de la cuadrícula.
column dense Coloca los elementos rellenando cada columna y rellena los orificios de la cuadrícula

CSS La propiedad grid-auto-rows

La propiedad grid-auto-rows establece un tamaño para las filas de un contenedor de cuadrícula.

Esta propiedad sólo afecta a las filas con el tamaño no establecido.

sintaxis CSS

grid-auto-columns: auto|max-content|min-content|length;
Value Description
auto Valor por defecto. El tamaño de las filas está determinado por el tamaño del elemento más grande en la fila
max-content Establece el tamaño de cada fila para que dependa del elemento más grande de la fila
min-content Establece el tamaño de cada fila para que dependa del elemento más grande de la fila
length Establece el tamaño de las filas, utilizando un valor de longitud legal. Lee sobre unidades de longitud
