HTML Vídeo

Multimedia en la web es: sonido, música, vídeos, películas y animaciones. Los archivos multimedia viene en muchos formatos diferentes. Puede ser casi cualquier cosa que puede escuchar o ver. Ejemplos: Imágenes, música, sonido, vídeos, discos, películas, animaciones y mucho más. Las páginas Web a menudo contienen elementos multimedia de diferentes tipos y formatos.

Etiqueta Descripción
<video> Define un video
<source> Define un archivo de texto
<track> Define pistas de texto en reproductores multimedia.
<audio> Define un audio
<iframe> Reproduce videos en HTML que usa YouTube.
<object> un complemento para ampliar la funcionalidad de un navegador web.
<embed> Los navegadores antiguos lo aceptan. Sin embargo, no ha sido ya no es parte de HTML5.


"Codigo fuente"

<video width="320" height="240" autoplay controls preload="auto">
<source src="video/mov_bbb.ogg" type="video/ogg">
<source scr="video/mov_bbb.mp4" type="video/mp4">
<track src="subtitulo_en.vtt" kind="subtitulos srclang="en" label="English">videos</track>
</video>
