Elementos de formulario HTML

El elemento <input>

El  elemento <input> se puede mostrar de varias maneras, dependiendo del type atributo.

Ejemplo: <input name="Nombre" type="text">

Si type se omite el atributo, el campo de entrada obtiene el tipo predeterminado: "texto".

El elemento <select>

define una lista desplegable :

Ejemplo

<select name="cars">
<option value="volvo">Volvo</option>
<option value="saab">Saab</option>
<option value="fiat">Fiat</option>
<option value="audi">Audi</option>
</select>

El elemento selecionado, define una lista desplegable

Los elementos <option> define una opción que puede seleccionarse.

De manera predeterminada, se selecciona el primer elemento de la lista desplegable.

Para definir una opción preseleccionada, añadir el selected atributo para la opción:

Ejemplo <option value="fiat" selected>Fiat</option>

Valores visibles:

Utilice el atributo size para especificar el número de valores visibles:

Ejemplo

<select name="cars" size="3">
<option value="volvo">Volvo</option>
<option value="saab">Saab</option>
<option value="fiat">Fiat</option>
<option value="audi">Audi</option>
</select>

Valores de opciones visibles

Utiliza el atributo tamaño para especificar el número de valores visibles.



Permite selecciones múltiples : (puedes marcar todos, con opción teclado)

Utilice el multiple atributo para permitir al usuario seleccionar más de un valor:

Ejemplo:

<select name="cars" size="4" multiple>
<option value="volvo">Volvo</option>
<option value="saab">Saab</option>
<option value="fiat">Fiat</option>
<option value="audi">Audi</option>
</select>

El Elemento <textarea>

El elemento <textarea> define un campo de entrada de líneas múltiples ( un área de texto ):

Ejemplo

<textarea name="message" rows="10" cols="30">
El gato juega en el jardín.
</textarea>

El atributo (rows) especifica el número visible de líneas en un área de texto.

El atributo (cols) especifica la anchura visible de un área de texto.

Así es como el código HTML anterior se muestra en un navegador:

También puede definir el tamaño del área de texto mediante el uso de CSS:

Ejemplo:

<textarea name="message" style="width:350px; height:100px;">
El gato juega en el jardín.
</textarea>

HTML <optgroup> Tag

Definición y Uso

La <optgroup> se utiliza para las opciones relacionadas con los grupos en una lista desplegable. Si usted tiene una larga lista de opciones, grupos de opciones relacionadas son más fáciles de manejar para un usuario.

atributos

Atributos Valor Descripción
disabled disabled Especifica que un grupo de opciones debe estar deshabilitado
label text Especifica una etiqueta para un grupo de opciones.

Ejemplo:

<select>
  <optgroup label="Swedish Cars">
    <option value="volvo">Volvo</option>
    <option value="saab">Saab</option>
  </optgroup>
  <optgroup label="German Cars">
    <option value="mercedes">Mercedes</option>
    <option value="audi">Audi</option>
  </optgroup>
</select>

HTML <label> Tag

Definición y Uso

La etiqueta <label> define una etiqueta para un botón <>, <input>, <metro>, <salida>, <progreso>, <seleccionar> o elemento <textarea>. El elemento <label> no hace como algo especial para el usuario. Sin embargo, proporciona una mejora la facilidad de uso para los usuarios de ratón, ya que si el usuario hace clic en el texto dentro del elemento <label>, se alterna el control. El atributo de la etiqueta <label> debería ser igual al atributo id del elemento relacionado para unirse juntos.

Consejo: Una etiqueta puede estar unido a un elemento ya sea usando el atributo "for", o colocando el elemento dentro de la <label> elemento.

Atributo Valor Descripción
for element_id Especifica a qué elemento de formulario está enlazada una etiqueta.
form form_id Especifica una o más formas a las que pertenece la etiqueta.

Ejemplo:

<form action="/action_page.php">
  <label for="male">Male</label>
  <input type="radio" name="gender" id="male" value="male"><br>
  <label for="female">Female</label>
  <input type="radio" name="gender" id="female" value="female"><br>
  <label for="other">Other</label>
  <input type="radio" name="gender" id="other" value="other"><br><br>
  <input type="submit" value="Submit">
</form>





HTML <select> Tag

Definición y Uso

El elemento <select> se utiliza para crear una lista desplegable. Las etiquetas <option>  dentro del elemento <select> definen las opciones disponibles en la lista.

Consejo: El elemento <select> es un control de formulario y se puede utilizar en una forma para recoger la entrada del usuario.
