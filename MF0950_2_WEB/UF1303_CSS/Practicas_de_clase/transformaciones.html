Las transformaciones CSS

transformaciones CSS le permiten trasladar, rotar, escalar y sesgar elementos. Una transformación es un efecto que permite un cambio de elemento de forma, tamaño y posición. CSS admite transformaciones 2D y 3D.

Transformaciones CSS 2D

En este capítulo aprenderá acerca de los siguientes métodos de transformación 2D:

  • translate()
  • rotate()
  • scale()
  • skewX()
  • skewY()
  • matrix()

El Método translación ()

El translate()método se mueve un elemento desde su posición actual (de acuerdo con los parámetros dados para el eje X y el eje Y).
