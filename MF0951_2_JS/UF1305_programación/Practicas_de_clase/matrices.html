Métodos numéricos

Método Descripción Síntasix  
new Array() Crea un nuevo objeto de tipo Array con los datos existentes en variables new Date()
split() Se utiliza para dividir una cadena en una matriz de subcadenas, y devuelve la nueva matriz. split()
concat() Une dos o más matrices y devuelve una copia de las matrices unidas concat()
copyWithin() Copia elementos de matriz dentro de la matriz, hacia y desde las posiciones especificadas copyWithin()
entries() Devuelve un par clave / valor Array Iteration Object entries()
toString() Convierte una matriz a una cadena de (separados por comas) valores de la matriz. toString()
join() Se une a todos los elementos de la matriz en una cadena. join()
pop() Elimina el último elemento de una matriz pop()
push() Añade un nuevo elemento a una matriz (al final) push()
shift() Elimina el primer elemento de matriz y "desplaza" todos los otros elementos a un índice inferior. shift()
unshift() Añade un nuevo elemento a una matriz (al principio) unshift()
length() Ofrece una forma fácil de añadir un nuevo elemento a una matriz length()
