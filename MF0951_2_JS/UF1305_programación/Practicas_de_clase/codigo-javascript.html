¿Dónde puede ser insertado JavaScript?

  • El código JavaScript en HTML debe ser insertado entre las etiquetas <script> y </script>.
  • Los scripts pueden ser colocados en la sección <body>, o en el <head> de una página HTML, o en ambos.
  • Los scripts también pueden ser colocados en archivos externos com js/funciones.js
  • El código JavaScript puede ser insertado en línea, en las etiquetas HTML, en el atributo de evento onclick.

Ejemplo de código JavaScript en la sección Head

<head>
<script>
function myFunction() {
 document.getElementById("demo").innerHTML = "Párrafo Cambiado.";
}
</script>
</head>

Ejemplo: JavaScript en el Body

<body> 

<h1>A Web Page</h1>
<p id="demo">A Paragraph</p>
<button type="button" onclick="myFunction()">Try it</button>
<script>
function myFunction() {
 document.getElementById("demo").innerHTML = "Paragraph changed.";
}
</script>
</body>
