Las posibilidades de visualización de JavaScript

JavaScript puede alterar los datos "Display" de diferentes maneras:

  • La escritura en un elemento HTML, usando  innerHTML.
  • La escritura en la salida HTML utilizando document.write().
  • Escribir en un cuadro de alerta, utilizando  window.alert().
  • Escribir en la consola del navegador, usando  console.log().

 

Usando innerHtml

Para acceder al contenido HTML de un elemento se utiliza innerHTML.

Usando document.write

Para propósitos de prueba se utiliza document.write() para mostrar los datos.

Usando window.alert

Para propósitos informativo utilizar el método window.alert() para mostrar los datos.

Usando console.log

Para propósitos de depuración utilizar el método console.log() para mostrar los datos.
