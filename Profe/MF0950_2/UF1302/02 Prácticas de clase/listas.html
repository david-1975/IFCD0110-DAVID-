Etiquetas de lista de datos

HTML ofrece varios mecanismos para especificar listas de información. Las listas se definen con las etiquetas <ul> , <ol>, <dl>. Todas las listas deben contener uno o más objetos de lista. Las listas pueden contener: información no ordenada, información ordenada, y lista de descripciones.

Etiqueta Descripción
<ul> Define una lista desordenada
<ol> Define una lista ordenada
<li> Define un elemento de una lista
<dl> Define una lista de definiciones
<dt> Define un término en una lista de definiciones
<dd> Define una descripción en una lista de definición

Tipos de listas

HTML ofrece varios mecanismos para especificar listas de información. Todas las listas deben contener uno o más objetos de lista. Las listas pueden contener: información no ordenada, información ordenada, información sobre definiciones.

Los tipos de listas en HTML son los siguientes:

Lista desordenada

  • Elemento 1
  • Elemento 2
  • Elemento 3
  • Elemneto 4

Lista ordenada

  1. Elemento
  2. Elemento
  3. Elemento
  4. Elemento

Lista definición

Titulo 1
Elemento 1
Título 2
Elemento 2

Lista anidada

  1. Elemento
    • Elemento 1
    • Elemento 1
  2. Elemneto

Nota: Los elementos de lista pueden contener otras listas, y otros elementos HTML, como imágenes y enlaces.

Listas desordenadas

Una lista no ordenada es un conjunto de elementos relacionados entre sí, sin ningún orden o secuencia determinados. La etiqueta <ul> define una lista desordenada y encierra todos los elementos. La etiqueta <li> encierra a cada uno de los elementos que componen la lista.

Lista type disc

  • Elemento 1
  • Elemento 2
  • Elemento 3
  • Elemneto 4

Lista type circle

  • Elemento 1
  • Elemento 2
  • Elemento 3
  • Elemneto 4

Lista type square

  • Elemento 1
  • Elemento 2
  • Elemento 3
  • Elemneto 4

Lista type none

  • Elemento 1
  • Elemento 2
  • Elemento 3
  • Elemneto 4

Nota: Se puede utilizar el estilos CSS "list-style-type" para especificar el tipo de viñeta. El atributo type en la etiqueta <ul> no se admite en HTML5. Los tipos de viñeta admintidas en CSS son: circle: círculo negro y relleno, disc: línea círcular, square: cuadro relleno, none: Este valor elimina la viñeta.

Listas Ordenadas

Las listas ordenadas son casi idénticas a las listas no ordenadas, salvo que en este caso los elementos relacionados se muestran siguiendo un orden determinado. La lista ordenada se define mediante la etiqueta <ol>. Los elementos de la lista se definen mediante la etiqueta <li>, la misma que se utiliza en las listas no ordenadas.

Lista type="1"

  1. Elemento 1
    1. Elemento 2
    2. Elemento 3
  2. Elemneto 4

Lista type="A"

  1. Elemento 1
  2. Elemento 2
  3. Elemento 3
  4. Elemneto 4

Lista type="I"

  1. Elemento 1
  2. Elemento 2
  3. Elemento 3
  4. Elemneto 4

Lista start="10" reversed

  1. Elemento 1
  2. Elemento 2
  3. Elemento 3
  4. Elemneto 4

Nota: Una lista ordenada puede ser numérica o alfabética. El atributo type en la entiqueta <ol> si se admite en HTML5 para cambiar el tipo de numeración. También se puede cambiar mediante el estilo CSS list-style-type. Los 3 atributos permitidos son"type" (tipo), "start" (comienzo) y "reversed" (invertido).

Listas de definición

Las listas de definiciones o descripciones son similares a las de un diccionario. Cada elemento de la lista está formado por un termino y una definición. La etiqueta <dl> define una lista de descripción, la etiqueta <dt> define el término (nombre), y la etiqueta <dd> describe cada término.

Lista con margen

Término 1
Definición 1
Término 2
Definición 2

Lista sin margen

Término 1
Definición 1
Término 2
Definición 2

Lista invertida

Término 1
Definición 1
Término 2
Definición 2

Lista dt, dd, dd, dd

Término 1
Definición 1
Definición 2
Definición 3

Nota: Los navegadores formatean las listas de definición de forma similar a las otras listas, tabulando la definición y alineando a la izquierda los términos. Aunque no es habitual, cada término puede tener asociada más de una definición y cada definición puede tener asociada varios términos.
